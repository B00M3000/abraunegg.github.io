OneDrive Client for Linux

A powerful and highly configurable OneDrive Client for Linux that is simple to use and runs on all major Linux distributions.

GitHub Installation Guide

Build from source or use a package manager

Get detailed installation instructions for your Linux distribution or build the OneDrive client from source code.

Docker Container

Quick deployment with Docker

Run the OneDrive client in a Docker container with pre-built images for multiple architectures including x86_64, ARM, and ARM64.

Features

Compatibility

Supports OneDrive Personal, OneDrive for Business, OneDrive for Office365, Sharepoint, and Azure National Cloud Deployments (Microsoft Cloud for US Government, Microsoft Cloud Germany, Azure and Office 365 operated by 21Vianet in China).

Distributions

Works on Debian, Ubuntu, Mint, Manjaro, SuSE, OpenSuSE, Fedora, CentOS, Red Hat, Arch and Slackware. across multiple architectures (i686, x86_64, ARMHF & ARM64)

Deployment

Can be built from source, Available as a package manager on official package managers. This OneDrive Client for Linux is also available as a Docker container utilising Alpine Linux, Debian Bullseye and Fedora, with pre-built images available for x86_64, ARMHF and ARM64.

Flexible Syncing

This OneDrive Client for Linux also supports one-way (upload or download only) and two-way sync operations with a variety of additional capabilities to suit your requirements and connects securely to Microsoft OneDrive services.

Data Integrity

Designed to ensure data integrity for your data - local state caching, file upload / download validation, resumable uploads

Caution

Test changes to your configurations to prevent accidental data loss.
